📌 Overview
Vox Poll is an advanced online polling system designed to make surveys, voting, and
opinion collection simple and secure. It provides a user-friendly interface, real-time results, 
and customizable poll options.

🚀 Features
Create and manage polls easily

Secure authentication for participants

Real-time vote counting and result visualization

Responsive design for mobile and desktop

Admin dashboard for managing polls and users

🛠️ Tech Stack
Html-for the body structure for the website 
css-for the frameworks and designing logic
javascript- for the code logic 
chart.js-for the chart live graph poll and the other graph making


📊 Usage
Register or log in to create polls

Share poll links with participants

View live results in the dashboard
